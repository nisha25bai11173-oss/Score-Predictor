Project Title

Student Score Prediction using Machine Learning

Description

This is a simple machine learning project that predicts a student's exam score based on the number of hours they studied. It uses Linear Regression, one of the most basic supervised learning algorithms. The goal is to understand how input (study hours) affects output (score).

Features

Simple dataset

Easy to understand code

Uses Linear Regression

Graph visualization included

Technologies Used

Python

NumPy

Pandas

Matplotlib

Scikit-learn

How It Works

The model is trained on a dataset containing study hours and corresponding scores. Once trained, it predicts the score for a new input (e.g., 9 hours of study).

How to Run

Install required libraries:

pip install numpy pandas matplotlib scikit-learn

Run the Python file:

python Score Predictor.py

Output

Predicted score displayed in console
Graph showing relationship between study hours and score
